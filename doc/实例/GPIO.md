# GPIO

GPIO设置分为三种设置

**NETLED**:此项可设置网络指示灯对应的GPIO引脚，用户根据实际需求，自行选择。

**NETRDY**:此项可设置 网络准备通知对应的GPIO引脚，用户根据实际需求，自行选择。

    LED 闪烁规则如下：
    100ms 闪烁 —— 注册GSM
    500ms 闪烁 —— 附着GPRS
    100ms 亮, 1900ms 灭 —— 已连接到服务器

**RSTCNF**:此项可设置硬件上**重置iRTU参数**对应的GPIO引脚，实际使用时，将该GPIO引脚拉低进行**重置iRTU参数**操作，用户根据实际需求，自行选择。
